# Toonify-Cartoon-Effect
A python implementation of the paper titled, "Toonify: Cartoon Photo Effect Application"

## Installation:

`$ pip install -r requirements.txt`

## Running the script:

`$ py toonify.py <optional_scaling_percent>`
